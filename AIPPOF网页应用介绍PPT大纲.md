### 第三部分：技术架构 (13-16页)

#### 第13页：核心算法详解

**T2税收优惠率**
```
公式：T2 = (实际税收节约 / 12000) × 100%

计算逻辑：
1. 逐年计算缴费带来的税收节约
   节约 = 缴费额 × 边际税率
2. 考虑投资回报率(1.75%)折现
3. 累加30年折现总节约
4. 除以上限12000转为百分比

典型值：
- 年薪¥80,000 → T2 ≈ 3-5%
- 年薪¥150,000 → T2 ≈ 10%
- 年薪¥300,000 → T2 ≈ 18%
```

**T3领取期税率**
```
公式：T3 = f₁(T2) + f₂(收入)

双逻辑斯蒂函数：
f₁(T2) = 7% / (1 + e^(-k₁×(T2-T2_mid)))
f₂(收入) = 7% / (1 + e^(-k₂×(收入-收入_mid)))

参数：
- L1 = 0%, L2 = 7%, L3 = 7%
- T2_mid = 5%, k1 = 2.0
- w_high = 500k, k2 = 0.00001

特点：
- 上限14%（远低于45%最高边际税率）
- 随T2和收入渐进增长
- 体现税收公平性
```

#### 第14页：补贴与上限机制

**精准补贴（两段式）**
```
第一段（首档）：
- 低收入（<8万）：45%配比
- 普通收入（≥8万）：30%配比

第二段（超额）：
- 统一6%配比

收入递减：
- ¥4万以下：全额补贴
- ¥4万-¥10万：线性递减
- ¥10万以上：补贴归零

示例：
年薪¥60,000, 缴费¥5,000
→ 补贴 = (5000×45% + 0×6%) × 削减系数
→ 补贴 ≈ ¥612/年
```

**动态上限（Formula 5-5）**
```
策略：min(动态上限, 固定上限)

固定上限：¥12,000/年

动态上限：f(年薪, T2率)
- 年薪¥60,000 → ¥8,000
- 年薪¥100,000 → ¥10,500
- 年薪¥150,000+ → ¥12,000（固定上限生效）

设计意图：
✅ 保护低收入：不限制合理缴费
✅ 激励中收入：允许适度缴费
✅ 约束高收入：防止过度税优
```
