## ⚠️ 发现的问题（2个严重问题）

### 🚨 **问题1：实际贴现率为负值**

**问题描述**：
```
实际贴现率 = 名义贴现率 - 工资增长率
            = 1.75% - 5.0%
            = -3.25%（负值！）
```

**影响**：
- 当工资增长率（5%）> 名义贴现率（1.75%）时，实际贴现率为负
- **未来现金流的实际价值被严重高估**
- 补贴的实际购买力被低估

**根本原因**：
- 补贴按照第0年收入计算，但未来30年工资会以5%/年增长
- 补贴金额未随工资增长而调整

**建议修正**：
1. 使用**实际贴现率**重新计算NPV：
   - 如果5%代表通胀，实际r = 1.75% - 5% = -3.25%
   - 如果5%仅是工资增长，应使用真实通胀率（如3%）
   
2. 补贴应随工资增长**动态调整**：
   ```python
   subsidy_t = subsidy_0 * (1 + wage_growth) ** t
   ```

3. 所有名义值转换为**实际购买力**（通胀调整）

---

### 🚨 **问题2：未考虑通胀对实际收益的侵蚀**

**问题描述**：
```
名义平均净收益:     ¥26,368
30年后实际购买力:   ¥10,863（假设3%通胀）
购买力损失:         58.8%
```

**影响**：
- 当前所有收益指标都是**名义值**，未反映实际购买力
- 30年后的¥26,368仅相当于今天的¥10,863
- **严重高估了实际经济效益**

**根本原因**：
- 模型中未引入通胀参数
- NPV计算使用名义贴现率，未区分名义vs实际

**建议修正**：
1. 引入通胀率参数（如3%）
2. 所有未来现金流转换为**实际价值**：
   ```python
   real_value_t = nominal_value_t / ((1 + inflation) ** t)
   ```
3. 使用**实际贴现率**：
   ```python
   real_r = (1 + nominal_r) / (1 + inflation) - 1
         = (1 + 0.0175) / (1 + 0.03) - 1
         = -1.21%
   ```

## 🔍 发现的改进点（2个次要问题）

### ⚠️ **改进点1：参与率函数可以更精细**

**当前问题**：
- 参与率仅基于净收益，未考虑收入水平本身
- 高收入者即使净收益相同，参与意愿也可能更低（有替代品）

**建议**：
引入收入弹性参数：
```python
def behavioral_participation_rate_v2(net_benefit, income):
    base_rate = f(net_benefit)  # 基础参与率
    income_adjustment = g(income)  # 收入调整系数
    return base_rate * income_adjustment
```

---

### ⚠️ **改进点2：T3税NPV应考虑死亡风险**

**当前问题**：
- 假设所有人在60-80岁都存活并缴纳T3税
- 未考虑死亡率对T3税收的影响

**建议**：
引入生存概率加权：
```python
for t in range(30, 50):
    survival_prob = life_table[age_30 + t]  # 查询生命表
    t3_tax_npv += t3_tax_t * survival_prob * discount_factor
```
